Detailed Description

Implementation of request handlers, responsible for initializing, sending and deleting a gRPC request. Applicable only for asynchronous requests.

Member Enumeration Documentation

enum GRPCClient::RequestHandler::RequestStatus

Request status. INIT means that it is initialized but not yet send. WAIT means that it has been sent but the reply is still pending.
