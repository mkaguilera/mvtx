Detailed Description

Generic class for request handlers, that are responsible for processing gRPC requests and replying to them.

Member Enumeration Documentation

enum GRPCServer::RequestHandler::RequestStatus

Request status for the server side. CREATE means that it is waiting for a request. PROCESS means that it has processed a request but has not yet sent the reply, and FINISH means that the reply has been sent and handler is available for garbage collection.
